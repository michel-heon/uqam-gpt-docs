# Guide de Démarrage Rapide - UQAM-GPT Postdoc

Commencez à utiliser UQAM-GPT Postdoc en 5 minutes !

## Installation dans Microsoft Teams

### Étape 1 : Accéder au Teams Store

1. Ouvrez **Microsoft Teams**
2. Dans la barre latérale gauche, cliquez sur **Apps**
3. Recherchez **"UQAM-GPT Postdoc"** dans la barre de recherche

### Étape 2 : Installer l'application

1. Cliquez sur la carte **UQAM-GPT Postdoc**
2. Cliquez sur le bouton **Ajouter** (ou **Add**)
3. Sélectionnez où installer :
   - **Pour vous** (conversation personnelle recommandée)
   - **À une équipe** (pour partager avec collègues)
   - **À un chat** (pour collaborer dans un groupe)

### Étape 3 : Premier lancement

Après installation, le bot vous envoie automatiquement un **message de bienvenue** avec :

- Présentation des 7 domaines de compétences
- Exemples de questions
- Divulgation IA obligatoire

## Première Utilisation

### Chat Personnel (1:1) - Recommandé pour débuter

1. Accédez à l'onglet **Chat** dans Teams
2. Trouvez **UQAM-GPT Postdoc** dans votre liste de conversations
3. Tapez votre première question :

```
Quelles sont les conditions d'admissibilité pour un postdoctorat ?
```

4. Le bot répond avec :
   - Réponse complète
   - Citations numérotées [1], [2], [3]
   - Liens vers sources UQAM
   - Boutons de feedback

### Chat de Groupe ou Teams

1. **Mentionnez le bot** avec `@UQAM-GPT Postdoc` suivi de votre question :

```
@UQAM-GPT Postdoc Quel est le montant du financement postdoctoral ?
```

2. Le bot répond uniquement quand vous l'appelez avec `@mention`

## Bonnes Pratiques

### À FAIRE

- **Soyez spécifique** : "Quelles sont les conditions d'admissibilité pour un chercheur étranger ?" plutôt que "Admissibilité ?"
- **Vérifiez les sources** : Cliquez sur les citations [1], [2] pour voir les documents sources
- **Signalez les problèmes** : Utilisez le bouton si une réponse est incorrecte
- **Contexte progressif** : Le bot se souvient de la conversation, posez des questions de suivi

### À ÉVITER

- **Questions trop vagues** : "Postdoc ?" → Trop général
- **Décisions officielles** : Le bot informe, mais ne prend pas de décisions administratives
- **Données personnelles sensibles** : Ne partagez pas de NAS, mot de passe, etc.
- **Informations confidentielles** : Le bot n'a pas accès aux dossiers individuels
- **Attentes de certitude absolue** : Vérifiez toujours les informations pour décisions importantes

## Comprendre les Réponses

### Format de Réponse Standard

```
[Réponse générée par IA avec contexte UQAM]

**Sources consultées :**
[1] Document UQAM - Lien
[2] Guide postdoctoral - Lien
[3] Politique financement - Lien

Cette réponse a été générée par UQAM-GPT, un agent d'intelligence 
artificielle générative basé sur GPT-4.1. Les informations proviennent 
de la documentation officielle de l'UQAM mais doivent être validées 
pour des décisions importantes.
```

### Citations Numérotées

Les **[1], [2], [3]** sont des liens cliquables vers les sources UQAM consultées.

**Pourquoi c'est important ?**

- Transparence : Vous voyez d'où viennent les informations
- Vérification : Vous pouvez lire le document source complet
- Confiance : Les réponses sont basées sur documentation officielle UQAM

### Streaming des Réponses

Le bot envoie sa réponse **progressivement** (mot par mot) pour une expérience fluide. Les citations apparaissent à la fin.

## Signaler un Problème

Après chaque réponse, vous verrez une **Adaptive Card** :

```
Cette réponse vous a-t-elle été utile ?

[Signaler un problème]  [Contacter le support]
```

### Quand signaler ?

- Réponse inexacte ou contradictoire
- Source citée incorrectement
- Contenu inapproprié
- Hallucination (invention d'informations)

**Action** : Cliquez sur **Signaler un problème**
→ Votre client email s'ouvre avec un template pré-rempli
→ Complétez la description du problème
→ Envoyez à Michel Héon

## Problèmes Courants

### Le bot ne répond pas

**Causes possibles :**

1. Dans un chat de groupe/équipe → Avez-vous utilisé `@UQAM-GPT Postdoc` ?
2. Service temporairement indisponible → Réessayez dans 2 minutes
3. Question trop vague → Reformulez avec plus de détails

### Citations ne s'affichent pas

- **Chat personnel** : Les citations apparaissent après le streaming (quelques secondes)
- **Chat de groupe** : Les citations sont intégrées directement dans le message

### Réponse incomplète

Si la réponse semble coupée :

- Le bot a atteint la limite de tokens
- Posez une question de suivi : "Peux-tu continuer ?" ou "Peux-tu développer le point 3 ?"

### Le bot répond hors-sujet

- Le contexte peut être mal interprété
- **Solution** : Reformulez votre question avec plus de précision
- Signalez le problème si cela persiste

## Sécurité et Confidentialité

### Ce que le bot SAIT

- Documentation UQAM publique
- Votre historique conversationnel (90 jours)
- Votre nom et ID Teams (AAD)

### Ce que le bot NE SAIT PAS

- Vos emails Teams/Outlook
- Vos fichiers SharePoint privés
- Dossiers candidatures individuelles
- Conversations avec d'autres personnes
