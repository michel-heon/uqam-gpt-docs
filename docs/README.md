# Documentation UQAM-GPT

Ce dossier contient les fichiers de documentation pour le wiki UQAM-GPT.

## Structure

- `wiki/` - Contenu du wiki GitHub

## Mise à jour du Wiki GitHub

Pour mettre à jour le wiki sur GitHub :

1. Clonez le dépôt wiki :
   ```bash
   git clone https://github.com/michel-heon/uqam-gpt-docs.wiki.git
   ```

2. Copiez les fichiers depuis `docs/wiki/` vers le dépôt wiki

3. Committez et poussez les changements :
   ```bash
   cd uqam-gpt-docs.wiki
   git add .
   git commit -m "Update wiki content"
   git push origin master
   ```

## Page d'accueil

La page d'accueil du wiki (`Home.md`) se trouve dans `docs/wiki/Home.md` et peut être mise à jour selon les besoins.
